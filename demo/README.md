# QAI Demo

This directory contains demonstration and reference materials for the QAI smart contract system on Qubic.

## Contents

| File | Description |
|------|-------------|
| `qrc20-example.md` | End-to-end example of creating and transferring a QRC-20 fungible token |
| `qrc721-example.md` | End-to-end example of NFT collection creation, minting, and transfer |
| `bonding-example.md` | End-to-end example of bonding curve token launch, buy/sell with slippage, and graduation |
| `contract-verification-checklist.md` | QA test vectors for all three contracts |

## How to Use

Each example markdown file contains:

- **RPC JSON payloads** — ready-to-use request/response pairs compatible with the Qubic JSON-RPC API
- **Step-by-step flow** — the expected sequence of calls for a realistic scenario
- **Expected results** — what each call should return or affect
