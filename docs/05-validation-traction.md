## Stage: Pre-build / Prototype

New project for a new ecosystem. Validation is based on:
- Ecosystem analysis of Qubic's current state

## Evidence

### 1. Pump.fun proved the bonding curve model

- **Solana Pump.fun:** Launched Jan 2024. Hit **$100M+ cumulative fees** within months.
- **How it works:** Bonding curve → trade directly → cap hit → auto-LP on DEX. Exactly QRC20-Bonding's model.
- **Why it worked:** Low barrier (30s to launch), instant tradability, fee on volume (not creation), viral share pages.
- **Signal:** Bonding curve platforms are the #1 growth engine for new tokens on any chain.

### 2. Qubic has no token infra at all

- No ERC-20 equivalent → no memecoins, no DeFi tokens
- No ERC-721 equivalent → no NFTs beyond QubicBay's custom contract
- No creation tools → every token requires C++ + PR + IPO
- **Signal:** The gap is total. First-mover advantage is absolute.

### 3. Builder track record

- Walytics won Walrus Cup 2026
- Sui Directory — 200+ projects indexed, live for months
- Aegis — on-chain reputation, live
- Qubic docs PR #79 — merged (typo fixes + NFT documentation)
- **Signal:** This team ships.
